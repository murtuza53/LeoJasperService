Recovery scenarios
  • Restore on a new machine: drop the two folders into the same paths, start the service, log in with the original admin password — every company keeps its keyId and continues to work.
  • Lost admin password: unrecoverable. AES-GCM + PBKDF2 has no backdoor. Bootstrap a fresh install (delete .leojasper/) and re-register companies.
  • Lost a single asset within 30 days: not a restore problem — admin can recover from the per-company .meta/trash/ via the company's row → Trash.
  • Lost an asset older than 30 days: needs a backup snapshot from before the sweep.
